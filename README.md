Bodega System - Backend

Sistema de gestión para bodegas, bares y comercios gastronómicos.

Funcionalidades

* Autenticación JWT
* Gestión de empresas (multiempresa)
* Gestión de productos
* Gestión de categorías
* Gestión de mesas
* Gestión de ventas
* Gestión de proveedores
* Gestión de facturas de proveedores
* Registro de pagos parciales y totales
* Cuenta corriente de proveedores
* Reportes 

Tecnologías

* Java 17
* Spring Boot
* Spring Security
* JWT
* Spring Data JPA
* MySQL
* Gradle

Seguridad

El sistema utiliza autenticación JWT.

Para acceder a los endpoints protegidos se debe enviar:

Authorization: Bearer TOKEN

Estructura principal

* auth/
* controller/
* entity/
* repository/
* service/
* security/

Estado del proyecto
